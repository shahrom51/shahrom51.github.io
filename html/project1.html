Codes Explanations
<!DOCType html> Every Html document starts with <!DOCType html>
<!DOCType html> <html> <head> <title></title> </head> <body> </body> </html> The Html document is divided into main parts enclosed in a pair of tags <html></html> followed by <head></head> and <body></body>
< head> <title></title> <style type="text/css"></style> </head> The head section contains the title and style. Inside the style tags, we type the Css styles that describe the look and feel of the contents.
<body> </body> Inside the tags of <body> </body> we put all the contents of the web page
<body> <div></div> <div></div> <div></div> <div></div> </body> We put different parts of contents inside different divisions of the page, inside pairs of <div></div>
<body> <div id="header"></div> <div class="panel"></div> <div class="panel"></div> <div id="footer"></div> </body> We can identify the div individually and uniquely by using the id selector, or classify them under the same class using the class selector. Later we use these selectors to assign style to each part of document.
